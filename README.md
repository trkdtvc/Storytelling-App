# Storytelling App

An interactive storytelling app built with HTML, CSS, and JavaScript. Users can choose between different story types (scary, funny, and adventure), and the app updates the displayed story content and visuals dynamically based on the selection.

## Features
- Story selection (scary, funny, adventure)
- Dynamic UI updates when a story is chosen
- Simple, responsive layout and styling
- Clear separation of structure (HTML), styling (CSS), and behavior (JavaScript)

## Tech Stack
- HTML
- CSS
- JavaScript

## Purpose
Built as a practice project to improve DOM manipulation, event handling, and dynamic UI updates with vanilla JavaScript.
